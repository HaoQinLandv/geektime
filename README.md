## 起因
极客时间特别不友好，付费购买后只能在移动端查看，我想PC端边做笔记边看，但是达不到，所以就自己动手把自己已购买的专栏爬下来，保存为html，音频保存为mp3，方便自己查看。

## 怎么做
利用findler抓包，手机端设置代理，可以看到是通过cookie形式传递，所以我们要做的就是记录他的url，记录自己的cookie，然后coding。

## 技术栈
java + okhttp + gson
